CSS Selectors

A cheat sheet for understanding the world of CSS selectors.

Selector Type CSS Format Selector Description
Element p Selects all elements of a specific type
Class .class-name Selects elements with a specific class attribute
ID #id-name Selects elements with a specific ID attribute
Universal * Selects elements with a specific ID attribute
Descendant div p Selects elements that are a descendant of a element
Pseudo-classes a:hover Selects elements based on their state or position
Child ul > li Selects elements that are a direct child of a specified element
Multiple elements , Use a comma to apply the same styling to multiple elements
